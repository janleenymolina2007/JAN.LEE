Pasos para crear este sitio web

  1. Crear un archivo llamado index.html.
  2. Usar estructura de menú a la izquierda y contenido a la derecha.
  3. Agregar las secciones directamente en el mismo archivo HTML.
  4. Usar botones o enlaces para mostrar u ocultar las secciones.
  5. Usar imágenes desde internet y un iframe de YouTube para el video.
